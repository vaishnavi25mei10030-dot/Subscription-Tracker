
Subscription Tracker

A Python-based console application that helps users manage and monitor their recurring subscriptions.
This project enables users to add, view, update, search, pause, resume, and delete subscriptions while automatically tracking due dates and highlighting upcoming renewals.

⭐ Overview

The Subscription Tracker is designed to solve the common problem of forgetting subscription renewal dates, leading to missed payments or unwanted charges.
Using a simple menu-driven interface, users can organize all their subscriptions in one place and easily check:

Subscription name

Amount

Billing cycle

Next due date

Days left until renewal

Status (Active/Paused)

This project was developed as part of the VITYARTHI – Build Your Own Project initiative to demonstrate modular programming, real-world logic implementation, and Python fundamentals.

⭐ Features

Add new subscriptions

View all subscriptions

Delete subscriptions

Pause or resume subscriptions

Update subscription amount

Search subscriptions by name

Highlight subscriptions due within 5 days

Automatically calculate days left for next due date
