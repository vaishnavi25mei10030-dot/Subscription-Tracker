
1. Project Title
Subscription Tracker
________________________________________
2. Overview of the Project
The Subscription Tracker is a Python-based console application developed to help users efficiently manage their recurring subscriptions. It allows users to add, view, update, search, pause, resume, and delete subscriptions.
The system automatically calculates how many days are left before the next due date and highlights subscriptions that are due soon.
________________________________________
3. Features
•	Add new subscriptions
•	View all subscriptions
•	Delete existing subscriptions
•	Pause or resume subscriptions
•	Change subscription amount
•	Search subscriptions by name
•	Highlight subscriptions due within 5 days
•	Automatic due-date countdown (days left)
________________________________________
4. Technologies / Tools Used
•	Python 3
•	datetime module
•	Command-line interface (CLI)
•	Modular programming structure (multiple .py files)
________________________________________
5. Steps to Install & Run the Project
Step 1 — Install Python
Download and install Python from the official website:
https://www.python.org/downloads/
Step 2 — Download the Project Files
Clone the GitHub repository or download the ZIP file containing the project folder.
Step 3 — Run the Application
The main menu will appear and you can begin using the system.
